# Student Management System (Java Swing)

## 📌 About Project
This is a Java Swing-based desktop application for managing student records.

## 🚀 Features
- Add Student
- View Students
- Search Student by ID
- Update Student Details
- Delete Student
- JTable for real-time data display
- Serial number auto-update

## 🛠 Tech Stack
- Java
- Swing (GUI)
- ArrayList
- JTable

## 🎯 Purpose
This project was built for learning Java GUI development and CRUD operations.
